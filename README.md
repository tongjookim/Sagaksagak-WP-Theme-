## 😤 개발 배경 (Why I Built This)

> "유료 테마 쓰다가 뒷목 잡고, 무료 테마 쓰다가 암 걸릴 뻔한 퍼블리셔 및 기획자가 **'빡쳐서'** 만들었습니다."

워드프레스를 좀 만져본 사람이라면 누구나 가슴 속에 사직서 대신 '직접 만든 테마' 하나쯤은 품고 살지 않나요?

* **유료 테마의 배신:** 슬라이더 하나 띄우려고 플러그인 50개를 깔아야 합니다. 내 서버는 비명을 지르는데, 개발자는 "Pro 버전을 사면 해결될지도?"라며 결제를 유도합니다.
* **무료 테마의 한계:** 가볍다고 해서 깔았더니 폰트 색상 하나 바꾸려 해도 'CSS 잠금'이 걸려 있습니다. 코드를 열면 스파게티처럼 꼬인 소스에 정신이 아득해집니다.
* **기획자의 비극:** "이 버튼 조금만 옆으로 옮겨주세요"라는 요청에 퍼블리셔가 테마 구조 때문에 안 된다며 울고 있습니다.

결국 "내가 쓸 테마, 내가 가장 가볍고 똑똑하게 만들자"는 일념으로 시작했습니다. 불필요한 기능을 시원하게 걷어내고, AI 시대에 발맞춰 누구나 프롬프트 몇 줄로 테마를 '조종'할 수 있는 가장 순수한 상태를 지향합니다.

---

## 💸 유료급 기능을 담았지만, 그냥 무료로 풉니다

"카카오/네이버 공유 API 연동, 채팅형 UI, 모바일 액션 그리드... 이거 다 구현하느라 피똥 쌌습니다."

솔직히 말씀드리면, 사각사각 테마에 들어간 기능들은 웬만한 유료 테마와 비교해도 꿀리지 않습니다.

* **API 풀 연동:** 카카오톡, 네이버 카페 등 국내 환경에 필수적인 공유 API가 이미 다 짜여 있습니다.
* **프리미엄 UI:** 유료 플러그인이 있어야 가능했던 **채팅형 댓글**과 **Glassmorphism 모바일 그리드**를 코드로 직접 구현했습니다.

이걸 왜 무료로 파냐고요? 저도 똑같이 빡쳐봤으니까요. 좋은 환경에서 같이 코딩하고 서비스 키워나가자는 마음으로 **100% 무료**로 공개합니다. 그냥 가져다 쓰세요. 그거면 됩니다.

## 🚀 프로젝트 철학

사각사각 테마는 단순한 스킨이 아니라, 순수 코드와 AI의 지능으로 채워진 플랫폼입니다.

1.  **플러그인 디톡스 (Zero-Bloat):** 수십 개의 플러그인 없이 가볍고 빠른 워드프레스 표준을 준수합니다.
2.  **AI 주도 개발 (AI-Native Architecture):** 생성형 AI(LLM)가 가장 이해하기 쉬운 구조로 설계되어, AI에게 코드를 던지면 최적의 커스텀 코드를 제안받을 수 있습니다.
3.  **노-업셀링 (No-Upsells):** "Pro 버전 업그레이드" 배너는 없습니다. 모든 설정은 사용자 정의하기(Customizer)에서 직관적으로 제어합니다.

## ✨ 핵심 기능 (Key Features)

### 💬 채팅형 댓글 UI (Messenger Style)
* 일반적인 댓글창 대신 카카오톡/라인 스타일의 말풍선 UI를 제공합니다.
* 작성자와 방문자의 말풍선을 좌우로 구분하여 생동감 넘치는 소통이 가능합니다.

### 📱 모바일 최적화 5열 그리드 버튼
* 하단에 **Glassmorphism** 스타일의 고정 액션 바를 제공합니다.
* 홈편집, 카테고리, 톡톡하기, 이웃추가, 통계 기능을 깔끔하게 배치했습니다.

### 🔗 지능형 SNS 공유 레이어
* 복잡한 공유 버튼을 숨기고 URL 복사와 카카오톡만 노출합니다. 나머지는 '더보기(⋯)' 레이어로 깔끔하게 관리합니다.

### 👤 빌트인 프로필 패턴
* 구텐베르크 블록 패턴으로 제작된 '프로필 페이지'를 포함합니다. 히어로 카드, Core Values 등을 손쉽게 커스텀하세요.

## 🛠 설치 및 시작하기

### 1. 저장소 클론
```bash
git clone [https://github.com/tongjookim/Sagaksagak-WP-Theme-.git](https://github.com/tongjookim/Sagaksagak-WP-Theme-.git)
```

### 2. 테마 업로드
워드프레스 관리자 화면의 외모 > 테마 > 새로 추가 > 테마 업로드에서 설치 후 활성화합니다.
(경로: ```/wp-content/themes/sagaksagak```)

🤖 AI로 테마 커스텀하기
이 테마의 진정한 위력은 AI 연동에서 나옵니다. 더 정교한 코드 수정을 위해 Claude Code, Gemini Code, OpenAI Codex 사용을 강력 권장합니다.

1. functions.php나 style.css 코드를 AI에게 전달합니다.

2. "사각사각 테마를 사용 중인데, [원하는 기능]을 추가하고 싶어"라고 프롬프트를 입력하세요.

3. 단순한 구조 덕분에 AI가 오류 없는 명확한 스니펫을 즉시 제공합니다.

**[!TIP]**
테마 내 포함된 sagaksagak_theme_cli.md 파일은 AI가 테마 전체 컨텍스트를 파악하는 데 최적화된 설계도입니다. 이 파일을 AI에게 학습시키면 훨씬 정확한 커스텀이 가능합니다.
